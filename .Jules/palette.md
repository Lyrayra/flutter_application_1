## 2024-05-24 - Add Password Visibility Toggle

**Learning:** When users enter passwords on mobile devices or tablets, typing errors are frequent due to the small touch keyboards. Hiding passwords without a way to reveal them can cause frustration and repeated connection errors in SSH settings. Adding a simple toggle allows users to verify their input while maintaining default security.
**Action:** Always include a visibility toggle (`Icons.visibility` / `Icons.visibility_off`) on password input fields, especially in configuration or settings screens.
